## 🛠️ Project setup

```bash
pnpm i
```

### 🔥 Compile and hot-reload for development

```bash
pnpm dev
```

### 🩺 Lint

```bash
pnpm lint
```

### 🔍 Type check

```bash
pnpm tsc
```

### 🧹 Format

```bash
pnpm format
```

### 🧪 Run tests

```
pnpm test

pnpm test:run

pnpm test:coverage
```

### 🚀 Compile and minify for production

```bash
pnpm build
```
